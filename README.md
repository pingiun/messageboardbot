# MessageBoardBot
On Telegram at @MessageBoardBot

## Installation
Install telepot with `pip install telepot`

## Running
1. Edit `config.ini` to include your bot token. (An example config file can be found in `config.sample`)
2. Run the bot with `python bot.py`
